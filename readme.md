# ng-style [![Build Status](https://travis-ci.org/ajoslin/ng-style.svg?branch=master)](https://travis-ci.org/ajoslin/ng-style)

> Write inline styles for Angular


## Install

```
$ npm install --save ng-style
```


## Usage

```js
var ngStyle = require('ng-style')

ngStyle('input')
//=> output
```
